# Go-Database-Client

## Introduction
* It's Database Client Package
* It provides creating connection pool, queries, and graceful shutdown

## Dependencies

### Postgres
```zsh
go get -u github.com/lib/pq
```

### Mariadb / Mysql
```zsh
go get -u github.com/go-sql-driver/mysql
```

## Installation

```zsh
go get github.com/donghquinn/gdct
```
